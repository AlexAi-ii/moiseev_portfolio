Бильярд бот

Бронирование столов и общение с клиентами в Telegram — быстро, удобно и автоматически

Описание проекта

Название проекта: бот для бронирования столов в бильярдном клубе «Игра».

Цель проекта: автоматизировать бронирование столов и улучшить клиентский сервис через Telegram‑бота.

Задачи

  • Подключение к базе данных и календарю для синхронизации занятости столов.
  • Вывод свободных столов в реальном времени.
  • Регистрация пользователей и бронирование через Telegram.
  • Уведомления о подтверждениях и напоминания.
  • Интеграция OpenAI для ответов на вопросы пользователей.

Платформа и интеграции

  • n8n — автоматизация логики и сценариев.
  • Google Calendar — синхронизация расписания столов.
  • Supabase — надёжная база данных.
  • OpenAI — ответы и голосовые сообщения на основе ИИ.

Сценарии использования

  • Пользователь запускает бота и получает всю необходимую информацию о клубе и ценах.
  • Проходит простую регистрацию, выбирает дату и время, бронирует стол и получает подтверждение.
  • Задает вопросы о клубе или правилах — бот быстро отвечает с помощью AI.
  • Администратор получает уведомления и управляет бронями через веб‑интерфейс.

Польза

Для клиентов: быстрое бронирование без звонков, возможность получать ответы прямо в Telegram.
Для бизнеса: современный канал продаж, снижение нагрузки на операторов, возможность собирать аналитику по посещаемости.
