Личный помощник через ТГ

Интеллектуальный помощник, который управляет задачами, встречами и письмами прямо из Telegram

Описание проекта

Описание: Интеллектуальный Telegram‑бот, который становится персональным ассистентом пользователя. Бот общается голосом и текстом, помогает планировать день, следит за встречами, ищет информацию, отправляет письма и работает с контактами — прямо из мессенджера.

Основной функционал

  • Голосовое и текстовое общение: распознавание голосовых команд (OpenAI Whisper) и поддержка текстового режима.
  • Работа с календарём: интеграция с Google Calendar; создание, редактирование и удаление событий; напоминания и планирование задач по голосу или тексту.
  • Контакты и адресная книга: доступ к Google Contacts (с разрешения пользователя); поиск и выбор контактов при отправке сообщений или приглашений.
  • Почта: интеграция с Gmail API для отправки писем, просмотра входящих, пересылок и автоответов.
  • Поиск информации в интернете: подключение к поисковым API для получения актуальных ответов; формирование краткой справки при помощи ChatGPT.

Технический стек

  • n8n — связующее звено для интеграций и автоматизации.
  • Telegram Bot API — интерфейс общения с пользователем.
  • OpenAI — генерация ответов и обработка данных (ChatGPT, Whisper).
  • Google API — Календарь, Gmail и Контакты.

Возможности расширения

  • Установка будильников и напоминаний на телефоне.
  • Отправка экстренных уведомлений или звонков по кодовому слову.
  • Подключение задач из Notion и Trello.
  • Управление умным домом и подключение платёжных систем.
