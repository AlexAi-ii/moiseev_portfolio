Отслеживание платежей

Система напоминаний и эскалации платежей для контроля дебиторки без ручной рутины

Описание проекта

Название проекта: бот для отслеживания платежей.

Цель проекта: автоматизировать напоминания о платежах, эскалировать просрочки и сохранять контроль над дебиторкой.

Задачи

  • Подключение к Google Sheets для хранения информации о клиентах и платежах.
  • Регулярная проверка таблицы и определение ближайших сроков оплат.
  • Отправка напоминаний клиентам через Telegram и WhatsApp.
  • Эскалация по стадиям: предупредить, напомнить настойчивее, уведомить менеджера.
  • Изменение статусов платежей и занесение результатов в таблицу.

Платформа и интеграции

  • n8n — организатор сценариев и эскалации.
  • Google Sheets — единая таблица платежей.
  • Telegram Bot API — уведомления клиентам.
  • Интеграция WhatsApp через API — напоминания по второму каналу.
  • Возможность расширения на SMS и email‑каналы.

Сценарии использования

  • Сервис ежедневно проверяет таблицу и определяет задолженности.
  • Бот отправляет напоминание в Telegram и, при необходимости, дублирует в WhatsApp.
  • Если клиент не оплатил, система уведомляет менеджера и переводит задолженность на следующий уровень эскалации.
  • Статус оплаты обновляется в Google Sheets и доступен для анализа.

Польза

Для клиентов: своевременные напоминания помогают не забывать об оплате.
Для бизнеса: прозрачный учёт долгов, автоматизация коммуникаций и сокращение просрочек.
