Авто‑публикация в Telegram

Сервис, который по ссылке создаёт контент и автоматически публикует его в ваш канал

Описание проекта

Описание: Система, позволяющая по одной ссылке подготовить полноценный пост: формирует статью, генерирует промпт для изображения и в несколько кликов публикует материал в Telegram‑канале. Пользователь контролирует процесс и утверждает результат перед публикацией.

Основные возможности

  • Импорт по ссылке: сервис принимает ссылку на статью или новость и извлекает ключевой контент.
  • Генерация статьи: формирует краткую и читабельную версию материала с помощью AI‑алгоритмов.
  • Создание изображения: генерирует описание для нейросети (DALL·E/Midjourney) и получает картинку, соответствующую теме.
  • Предпросмотр и согласование: отправляет пользователю черновик текста и картинки для одобрения или правки.
  • Автопубликация: после утверждения пост автоматически появляется в Telegram‑канале, включая прикреплённое изображение.

Технический стек

  • n8n — автоматизация шагов и интеграций.
  • ChatGPT / OpenAI — анализ текста и генерация статьи.
  • Модели генерации изображений — создание иллюстрации на основе промпта.
  • Telegram Bot API — размещение постов в канале.

Галерея

Сервис можно адаптировать под разные форматы: новости, статьи, подборки. Готов показать прототип и обсудить интеграцию под ваш канал.
